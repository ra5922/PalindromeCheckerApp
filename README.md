Step 1: Start
Step 2: Input

Read a string s from the user.

Step 3: Find Length

Find the length of the string and store it in variable n.

Step 4: Initialize

Set a boolean variable a = true (assume it is a palindrome).

Step 5: Compare Characters

Repeat for i = 0 to n/2 - 1:

Compare s[i] with s[n - 1 - i]

If they are not equal:

Set a = false

Print "Not a Palindrome"

Stop the loop

Step 6: Check Result

If a == true

Print "It is a Palindrome"

Step 7: End
